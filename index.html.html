The 3 Iron Rules
🎯
One Task Per Day
Pick ONE business task each morning. Do not end the day without completing it.
🔁
Never Miss Twice
Miss a habit once — fine. Miss twice in a row — you're building a new bad habit.
📵
No Phone Before Work Block
Phone check before your work block poisons your focus for hours. Earn it after.
Your Daily Non-Negotiables
BLOCK 1 · 30 MIN
Morning Sharp
  • Face wash + moisturiser + SPF (5 min)
  • Fix hair intentionally — not just run hands through it
  • Outfit chosen night before — wear with intention
  • Journal: write the ONE task you will complete today
  • No phone until this block is done
BLOCK 2 · 60–90 MIN
Deep Work
  • ONE task only — the one from the journal
  • Timer on. Phone face-down or in another room
  • No tabs unrelated to the task
  • Current focus: n8n automation productisation
  • Work until done or timer ends — no exceptions
BLOCK 3 · 30 MIN
Body + Review
  • 20 min movement — gym, walk, home workout (any)
  • Did you complete the one task? Yes = win. Log it.
  • No = what blocked you? Write it. Fix it tomorrow.
  • Lay out tomorrow's outfit
  • Rate today 1–10. Aim to average 7+ over 90 days
